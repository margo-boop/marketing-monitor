ШАГ 1. Распакуйте архив (zip) — кликните по нему два раза.
        Появится папка "marketing-monitor".

ШАГ 2. Зайдите в эту папку и запустите файл под вашу систему:

   ▸ На Mac      →  дважды кликните  ЗАПУСТИТЬ-MAC.command
   ▸ На Windows  →  дважды кликните  ЗАПУСТИТЬ-WINDOWS.bat

   Программа сама всё проверит, при необходимости доустановит
   нужное и через пару секунд откроет сайт в браузере.

ШАГ 3. В браузере откроется страница (адрес http://127.0.0.1:8787).
        Выберите месяц и нажмите "Собрать отчёт".
        Первый сбор идёт 1–2 минуты — это нормально.

────────────────────────────────────────────────────────────
ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК
────────────────────────────────────────────────────────────

• Mac пишет "неизвестный разработчик / нельзя открыть":
   кликните по ЗАПУСТИТЬ-MAC.command ПРАВОЙ кнопкой мыши →
   "Открыть" → ещё раз "Открыть". Так нужно только один раз.

• Пишет, что нет Python:
   Mac — появится окно установки Apple, нажмите "Установить".
   Windows — откроется сайт python.org, установите Python и
   ОБЯЗАТЕЛЬНО поставьте галочку "Add Python to PATH".
   После установки снова запустите файл.

• Страница не открылась сама:
   откройте браузер и вручную введите:  http://127.0.0.1:8787

• Остановить программу:
   просто закройте чёрное окно терминала (или Ctrl+C в нём).
