КАК ЗАПУСТИТЬ

1. Откройте файл OPEN_MONITOR.command
2. Дождитесь открытия страницы в браузере
3. Выберите месяц
4. Вставьте выгрузку из SpyWords / Вордстата, если она есть
5. Если есть API token или ссылка экспорта, вставьте их в блок "Если есть API или ссылка экспорта"
6. Нажмите "Собрать отчёт"
7. Внизу появится кнопка "Открыть PDF"

ВАЖНО

Сайты и Telegram собираются автоматически.
SpyWords и Вордстат закрыты логином, поэтому точные цифры нужно вставить выгрузкой или вручную.
Если есть API token или ссылка экспорта, вставьте их на странице в соответствующие поля.
Система не придумывает данные.
