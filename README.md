# Мониторинг конкурентов

Простой запуск для человека без технических знаний.

## Как пользоваться

1. Откройте файл `OPEN_MONITOR.command`.
2. Дождитесь, пока откроется страница в браузере.
3. Выберите месяц отчёта.
4. Вставьте выгрузку из SpyWords / Вордстата в большое поле, если она есть.
5. Если есть API token или ссылка экспорта, вставьте их в блок `Если есть API или ссылка экспорта`.
6. Нажмите `Собрать отчёт`.
7. Внизу страницы появятся кнопки `Открыть PDF`, `Открыть отчёт`, `Данные JSON`.

## Что собирается автоматически

- публичные страницы сайтов конкурентов;
- новости, статьи, кейсы и ссылки с сайтов;
- картинки с сайтов;
- публичные посты Telegram, если указан канал;
- готовый HTML-отчёт;
- готовый PDF.

## Что нужно вставлять вручную

SpyWords и Вордстат закрыты логином, поэтому точные цифры оттуда нужно вставить выгрузкой или вручную.

Примеры строк:

```text
чек офис ; 1631
check office ; 920
CheckOffice Яндекс TOP-50 312
CheckOffice Яндекс TOP-10 41
CheckOffice Google TOP-50 280
MD Audit Telegram 1250
```

Если выгрузки нет, отчёт всё равно соберёт публичную активность, но закрытые SEO-цифры останутся пустыми. Система не придумывает данные.

## Если есть API token или ссылка экспорта

На странице есть блок `Если есть API или ссылка экспорта`.

Заполните только то, что есть:

- `API token` — ключ доступа к сервису, если его выдали.
- `Ссылка экспорта SpyWords` — ссылка на выгрузку или страницу экспорта SpyWords.
- `Ссылка экспорта Вордстат` — ссылка на выгрузку или страницу экспорта Вордстата.
- `Дополнительные страницы` — любые страницы, которые нужно дополнительно прочитать, через запятую.

После заполнения нажмите `Собрать отчёт`.

## Если нужно поменять сайты или Telegram

На странице есть раскрывающийся блок `Сайты и Telegram-каналы конкурентов`.

Сайты уже заполнены. Telegram можно добавить в формате:

```text
@channel
https://t.me/channel
```

Доступы не сохраняются в проекте. Если сервис просит капчу, двухфакторный код или запрещает парсинг, система покажет ошибку и не будет подставлять выдуманные цифры.
