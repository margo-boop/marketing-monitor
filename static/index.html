API или ссылка экспорта

Куда заходить: SpyWords → домен → Экспорт; Яндекс Вордстат → запрос → копируйте цифры.

Откуда: spywords.ru → домен → Экспорт

Откуда: wordstat.yandex.ru → выгрузка

Где взять API token и ссылку экспорта +
КОРОТКО: чаще всего API token НЕ нужен. Самый простой путь — открыть сервис, скопировать цифры и вставить их в большое поле выше. ──────────────────────────────────── SpyWords — https://spywords.ru/ 1. Войдите в аккаунт (нужен платный тариф для API/экспорта). 2. Введите домен конкурента, например checkoffice.ru. 3. Вкладки «Запросы Яндекс / Google» → кнопка «Экспорт». 4. API token: правый верхний угол → «Настройки» → «API». ──────────────────────────────────── Яндекс Вордстат — https://wordstat.yandex.ru/ 1. Войдите под Яндекс-аккаунтом. 2. Введите запрос, например «чек лист аудит». 3. Скопируйте строки вида: чек офис ; 1631 check office ; 920 и вставьте в большое поле выше. ──────────────────────────────────── ИТОГ: если нет платного API — оставьте поля пустыми, вставьте цифры текстом и нажмите «Собрать отчёт».
