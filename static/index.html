Мониторинг конкурентов

Выберите месяц, вставьте данные или API-доступ и нажмите одну кнопку. Сайты и Telegram собираются автоматически.

1. МесяцВыберите месяц отчёта.
2. ДанныеВставьте выгрузку, API token или ссылку экспорта.
3. КнопкаНажмите «Собрать отчёт» и откройте PDF.

Готовый результат

После запуска появятся готовые файлы:

PDF для отправки, HTML-отчёт для просмотра в браузере и JSON с собранными данными.

Если нет API или выгрузки, отчёт всё равно соберёт публичную активность. Закрытые SEO-цифры появятся только после вставки настоящих данных.

Данные для отчёта

Можно вставить выгрузку текстом или подключить API/ссылку экспорта. Если ничего нет, оставьте поля пустыми и нажмите кнопку сбора.

Если есть API или ссылка экспорта

Заполните только то, что есть. Зайдите в сервис, сделайте экспорт/выгрузку, а ссылку или API token вставьте сюда. После этого нажмите «Собрать отчёт» сверху или снизу.

Куда заходить за данными:
  • SpyWords — spywords.ru → откройте домен конкурента → «Экспорт» → вставьте ссылку или скопированные цифры.
  • Яндекс Вордстат — wordstat.yandex.ru → введите запрос → выгрузите показы → вставьте ссылку или цифры.

Откуда брать: spywords.ru → домен → Экспорт.

Откуда брать: wordstat.yandex.ru → запрос → выгрузка.

Где взять API token и ссылку экспорта (пошагово)
КОРОТКО: чаще всего API token НЕ нужен. Самый простой путь — открыть сервис, скопировать цифры и вставить их в большое поле выше или во вкладку «Ввести цифры вручную». API token нужен только если у вас платный доступ с API. ──────────────────────────────────── SpyWords — https://spywords.ru/ 1. Войдите в аккаунт (нужен платный тариф для API/экспорта). 2. Введите домен конкурента, например checkoffice.ru. 3. Вкладки «Запросы Яндекс / Google» → кнопка «Экспорт» (выгрузка в Excel/CSV). • Можно просто выделить таблицу и скопировать цифры в большое поле выше. 4. API token: правый верхний угол → «Настройки аккаунта» → раздел «API». Скопируйте ключ и вставьте его в поле «API token». Прямой ссылки экспорта обычно нет — берите цифры из выгрузки. ──────────────────────────────────── Яндекс Вордстат — https://wordstat.yandex.ru/ 1. Войдите под Яндекс-аккаунтом. 2. Введите запрос, например «чек лист аудит», выберите регион. 3. Посмотрите число показов в месяц слева. 4. У Вордстата НЕТ простого API-токена для обычного пользователя. Просто скопируйте строки вида: чек офис ; 1631 check office ; 920 и вставьте в большое поле выше. Этого достаточно. (Полноценный API — только через Яндекс Директ OAuth, это для разработчиков.) ──────────────────────────────────── ИТОГ: если нет платного API — оставьте поля API token / ссылок пустыми, вставьте цифры текстом и нажмите «Собрать отчёт». Сайты, Telegram, соцсети, картинки и email соберутся автоматически без всякого токена.
